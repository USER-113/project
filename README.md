Проект "Анализ динамики атаки БПЛА территорий государства".

Целью данного проекта являлся сбор сообщений с телеграм канала МО РФ с информацией по уничтоженным дронам над территорией старны. И построение небольшого отчета с отображением динамим по субъектам государства.
Данные выложенные в проекте располагаются в одном месте, это папкf c DAG файлами программы Apache Airflow.

Сообщения собираются с телеграм канала: https://t.me/mod_russia

Канал официально включен в реестр роскомнадзора:

Официальный канал Минобороны России

В реестре Роскомнадзора: https://clck.ru/3FVQPX

Сообщения выбираются с орпеделенным содержимым, затем разбиваются на строки по датам и по субъектам, если указано число для перечисленных регионов без конкретной детализации по регионам, то общее количество делится на все регионы.

Итак, все выложенные тут файлы и папка должны лежать там, где запускаются dag файлы для Airflow.
